Conexión con GitHub

Para que el panel publique cambios necesita un token de GitHub (se guarda solo en este navegador).

  1. Entra a github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens.
  2. Generate new token → acceso solo al repo gearup_motors → permiso Contents: Read and write.
  3. Copia el token (empieza con github_pat_…) y pégalo aquí abajo.
